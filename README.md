# 🏏 India vs South Africa – T20 World Cup 2024  
## 📊 Performance Analytics Dashboard (Power BI)

---

## 📌 Project Overview

This project presents an interactive Power BI dashboard analyzing the India vs South Africa match from the ICC Men's T20 World Cup 2024.

The objective of this analysis is to transform raw match-level cricket data into structured performance insights using data modeling, DAX measures, and interactive visualization techniques.

This project demonstrates practical data analytics skills applied in a sports performance context.

---

## 🎯 Problem Statement

How can match data be converted into meaningful insights to evaluate:

- Batting efficiency?
- Strike rate effectiveness?
- Boundary impact on total score?
- Bowling economy and consistency?
- Individual player contribution?

---

## 📊 Key Performance Indicators (KPIs)

- Total Runs Scored
- Balls Faced
- Strike Rate
- Number of 4’s & 6’s
- Bowling Economy Rate
- Player Contribution Analysis
- Over-by-Over Run Distribution
